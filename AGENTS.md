# Blink Agent Guide

Scope: the `Blink` repository.

## Purpose

Blink is the deploy, verify, rollback, and reporting CLI for manifest-driven operations.

## Workflow

- Keep the command surface and manifest behavior truthful in `README.md` and `BLINK.md`.
- Keep active unfinished work in the workspace root `ROADMAP.md`.
- Record meaningful product or implementation changes in `CHANGELOG.md`.

## Validation

Prefer the local Ruby test suite and command checks for validation.
